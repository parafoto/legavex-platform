# LegaVex Web Frontend

> ⚠️ **Заглушка** - фронтенд будет реализован на Этапе 2

## Планируемый стек

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Auth:** NextAuth.js
- **State:** React Query / SWR

## Планируемые страницы

### Кабинет консультанта

- `/consultant/login` - Вход
- `/consultant/dashboard` - Дашборд с табами
- `/consultant/cases` - Список дел
- `/consultant/cases/[id]` - Карточка дела
- `/consultant/profile` - Профиль

### Админ-панель

- `/admin/dashboard` - Все дела
- `/admin/cases/[id]` - Карточка дела + аппрув
- `/admin/settings` - Глобальные настройки (тоглы)
